This repository is an exercise for my DWDWM course with the Centre Européen de Formation, in which I create an HTML base of my personnal CV using given indications, and style it using CSS.
The objective is to ensure all instructions are respected, as well as SEO recommendations and the use of Github repo for the entire project: new issue > new branch > commits > pull requests > merge > start again
What follows is a summary of the instructions, in french : 

Mission et Importance du CV ​

Créer un CV en ligne permet de présenter son parcours scolaire et professionnel tout en démontrant des compétences techniques en HTML, CSS et GitHub. ​
Un CV en ligne se distingue par sa capacité à mettre en avant des compétences variées, ce qui est apprécié par les recruteurs. ​

Identité Graphique

Utilisation de deux polices : Lato pour les titres et Raleway pour le contenu, avec importation via Google Fonts et @font-face. ​
La photo de profil doit être personnelle et arrondie, avec un favicon et des icônes de réseaux sociaux provenant de Flaticon ou Font Awesome. ​
Le CV doit respecter une palette de couleurs définie.

Livrable et Contenu

Le livrable est un lien vers un repository GitHub public, structuré avec des sous-dossiers.
La page doit inclure des informations classiques d'un CV, telles que les coordonnées, compétences, expériences, et liens vers les réseaux sociaux. ​
Des effets graphiques doivent être intégrés pour dynamiser le CV, comme des changements de couleur au survol d'éléments.
